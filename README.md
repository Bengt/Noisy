Noisy is a jQuery plugin that adds random noise to a given element.

#### Here's an example:
    $('body').noisy({
        noiseParticles: 40000, 
        tileSize: 200, 
        backgroundColor: '#EDEBDE', 
        maxNoiseOpacity: 0.078
    });

You can try a demo [here](http://jsbin.com/awole4).
